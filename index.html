// 01. About

Who I Am

I'm Kartik Kumar Singh, an Electronics & Communication Engineering student with a deep passion for embedded systems, robotics, and applied AI. I love building things that exist in the real world — hardware that thinks, robots that see, and systems that act.

My work lives at the intersection of firmware and intelligence — from training YOLOv8 models and deploying them on Raspberry Pi to designing PCBs in KiCad and printing custom enclosures. I'm equally comfortable in a Python IDE, a circuit schematic, or a 3D modeling tool.

I'm driven by the challenge of building smart, low-cost, real-world solutions — technology that doesn't just run in the cloud, but runs on the edge, in the field, where it matters most. My long-term goal is to contribute to cutting-edge AI and robotics research at NVIDIA.

// 03. Projects

What I've Built

PROJECT 001

Smart Farming Robot – Crop Detection & Handling

A low-cost prototype that detects and handles crops automatically using computer vision and robotic arm control. Tested with a pencil as reference object before scaling for real agricultural use. Runs an ONNX-optimized YOLOv8 model directly on a Raspberry Pi 4 with live web streaming via Flask.

  • YOLOv8 → ONNX model for edge deployment on Raspberry Pi
  • Servo-controlled robotic arm for pick-and-place automation
  • Live Flask web streaming for remote monitoring
  • L293D motor driver for multi-motor coordination
Python YOLOv8 ONNX OpenCV Flask Raspberry Pi 5 Servo Motors
PROJECT 002

Smart CCTV – AI Crowd Monitoring & Stampede Prevention

An intelligent edge-computing surveillance system that monitors crowd density in real time and prevents dangerous situations like stampedes. The AI model runs locally on Raspberry Pi 5 — no cloud dependency — and triggers multi-level GSM alerts to authorities as crowd density escalates.

  • Real-time crowd density estimation with AI (Normal / Increasing / High-Risk)
  • Stepped alert system via GSM — notifies escalating authority levels
  • Edge inference on Raspberry Pi 5 — zero cloud dependency
  • Immediate emergency trigger for abnormal crowd spikes
Python Computer Vision Raspberry Pi 5 GSM Module Edge AI OpenCV
